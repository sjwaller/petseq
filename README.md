# PETSeq v1.1
Step Sequencer for Commodore PET 4032

Based on original PETSynth - by Chiron Bramberger  www.petsynth.org

```r: randomize notes
RUN/STOP: quit
space: start/stop
m: mode (loop/bounce)
z: decrease bpm
x: increase bpm
c: decrease bpm x 10
v: increase bpm x 10
e: edit mode
n: step forward
b: step back
t: test note
Valid note values are: 0 - 100
```
